What is Python?

Python is a high-level, interpreted, general-purpose programming language known for its simplicity, readability, and versatility. Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages worldwide.

Easy to Learn

Clean syntax reduces learning curve for beginners

High Productivity

Rapid development with built-in libraries

Versatile

Used for web dev, data science, ML, automation, and more

Python Tokens

Tokens are the smallest individual units in a Python program. They include keywords, identifiers, literals, operators, and punctuation.

1. Keywords

Reserved words that cannot be used as identifiers

if else for while def class import return True False None import pass lambda try with

2. Identifiers

Names given to variables, functions, classes, etc.

# Valid Identifiers
my_variable = 10
UserProfile = "Aryan"
_private_var = "secret"

# Invalid Identifiers
# 123var (starts with number)
# var-name (contains hyphen)
# class (keyword)

3. Literals

Fixed values that appear in code

# Numeric Literals
integer = 42
float_num = 3.14
complex_num = 2 + 3j

# String Literals
single_quote = 'Hello'
double_quote = "World"

4. Operators & Punctuation

Symbols that perform operations or structure code

+ - * / == != () [] {} : ,

Python Data Types

Python has built-in data types that define the kind of value a variable can hold. They are categorized into several groups:

Numeric Types

  • int: Integer values (e.g., 10, -5, 0)
  • float: Floating-point numbers (e.g., 3.14, 2.5)
  • complex: Complex numbers (e.g., 2+3j, 5-2j)
age = 25
pi = 3.14159
z = 10 + 2j

Sequence Types

  • str: String (text data, e.g., "Hello")
  • list: Ordered, mutable collection (e.g., [1,2,3])
  • tuple: Ordered, immutable collection (e.g., (1,2,3))
name = "Alice"
fruits = ["apple", "banana"]
coords = (10.5, 20.3)

Mapping & Set Types

  • dict: Key-value pairs (e.g., {"name": "Bob"})
  • set: Unordered, unique elements (e.g., {1,2,3})
  • frozenset: Immutable set (e.g., frozenset({1,2}))
user = {"name": "Bob", "age": 30}
unique_numbers = {1,2,3,3,4}

Other Types

Boolean

Represents truth values: True or False

is_active = True
is_verified = False

None Type

Represents absence of value: None

result = None
print(result) # Output: None

Control Flow Basics

Control flow statements determine the order in which code is executed. Python supports conditional statements and loops:

Conditional Statements (if-elif-else)

Execute code based on conditions

age = 18

if age >= 18:
    print("You are an adult")
elif age >=13:
    print("You are a teenager")
else:
    print("You are a child")

For Loop

Iterate over sequences (lists, strings, etc.)

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)

# Range loop
for i in range(5):
    print(i)

While Loop

Execute code repeatedly as long as a condition is true

count = 0

while count < 5:
    print(count)
    count += 1

# Break and continue statements
for i in range(10):
    if i == 5:
        break # Exit loop
    if i %2 ==0:
        continue # Skip iteration
    print(i)
